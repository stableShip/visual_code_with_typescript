# visual_code_with_typescript
visual stdio code 的typescript项目模板

## 环境依赖
nodejs

typescript

tsd


##安装相关的依赖
npm install

##安装d.ts文件
1.npm install tsd -g
2.tsd install

##运行项目
1.安装typescript
>npm install typescript -g

2. 使用visual stdio code 打开项目,打开要编译的文件
   使用快捷键 `shift + alt + v`
   即可编译ts文件

命令行编译:
`tsc --module commonjs app.ts`
